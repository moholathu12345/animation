Guided Learning + Project Background

Guided Learning is a self-directed module where I planned a learning project, gathered evidence, monitored progress, and reflected using Plan–Perform–Monitor–Reflect.

For this project, I focused on learning 3D animation in Blender by creating a short dance animation. I used reference analysis, key poses, timing refinement, beat synchronization, camera framing, and lighting to improve presentation quality.

Reflection

Plan, Perform, Monitor, Reflect

Plan

I planned my learning using weekly milestones and chose a dance animation so I could apply timing, posing, and camera choices in a single focused output.

  • Went well: clear milestones kept me organised
  • Did not: underestimated time needed for polish
  • Action: split work into blocking, refine, polish passes

Perform

I built key poses first, refined transitions, synced movement to beats, then improved presentation using camera angles and lighting.

  • Went well: beat sync improved rhythm and structure
  • Did not: some transitions looked floaty initially
  • Action: adjusted timing, spacing, and reworked key poses

Monitor

I reviewed playblasts regularly, compared against reference, and tested rendering early to catch issues with ambience and background.

  • Went well: frequent previews caught issues quickly
  • Did not: technical blockers slowed progress at times
  • Action: changed one thing at a time, tested, then saved

Reflect

I learned that strong animation comes from process and iteration. This project improved my confidence in learning difficult tools through structured practice.

Resilient Inquisitive Communicates Effectively

Next time, I would plan shots earlier, refine arcs/spacing more, and make lighting choices more intentional.
